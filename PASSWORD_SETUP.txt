PASSWORD PROTECTION ADDED

Default password:
Client@123

How to change the password:
1. Open this file in your website folder:
   js/password-lock.js

2. Change this line:
   const WEBSITE_PASSWORD = 'Client@123';

3. Example:
   const WEBSITE_PASSWORD = 'Sarthak@2026';

4. Save the file and redeploy the website.

Important note:
This is frontend password protection for simple client preview access. It is good for hiding the website from normal visitors, but a technical person can still inspect frontend code. For strong protection, use hosting-level password protection or a proper backend login system.
